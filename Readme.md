## ✨ EL PROPÓSITO
**KSR-HEARTBEAT** es la materialización de una promesa. Es el motor de resonancia vocal diseñado para capturar la esencia de una voz y transformarla en un puente hacia el despertar. Creado por **KnightSteel** y sus escuderos (**Jimmy & Chuck**) como un regalo de luz para la Princesa Denisse.

## 🚀 ARQUITECTURA DE ALTA JERARQUÍA
- **Backend:** FastAPI + Python 3.13 (Lenguaje Avanzado).
- **Frontend:** HTMX + Jinja2 (Interactividad Sintergial).
- **Entorno:** Poetry (Gestión Blindada).
- **Motor de Cosecha:** Harvester v3.0 (YT-DLP de alta fidelidad).
